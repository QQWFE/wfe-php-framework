Tencent业务相关

获取基于微信的js-sdk：

需要在配置文件中对业务的app_id和app_secret进行配置 


'tencent' => array(
	'app_id' => '[your_app_id]',
	'app_secret' => '[your_app_secret]',
),

在实际取用的过程中，可以通过框架内置的类库功能进行选用


require_once ROOT_PATH."/core/Tencent/Jssdk/jssdk.php";
$jssdk = new JSSDK(Config::get_config_value('tencent')['app_id'], 
	Config::get_config_value('tencent')['app_secret']);
$sign_package = $jssdk->GetSignPackage();

输出的$sign_package即为有效的微信sdk验证token
